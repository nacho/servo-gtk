# Servo GTK

A GTK4 widget that embeds the Servo web engine.

## Features

- GTK4-based web browser interface
- Servo web engine integration
- OpenGL-accelerated rendering
- Async event handling

## Building

```bash
cargo build
```

## Running

```bash
cargo run
```

## Dependencies

- GTK4
- OpenGL
- Servo web engine
- Rust toolchain
